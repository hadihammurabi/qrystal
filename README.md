# qrystal

This is a query (SQL) builder for [Crystal Language](https://crystal-lang.org)

## Installation

1. Add the dependency to your `shard.yml`:
```yaml
dependencies:
  qrystal:
    github: hadihammurabi/qrystal
```
2. Run `shards install`
